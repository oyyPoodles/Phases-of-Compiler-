# Compiler Atlas: 6 Phases Explained

A high-fidelity, interactive educational tool built with React and Vite to visualize the journey of a C program through the various phases of a compiler.

## 🚀 Features

- **Interactive Pipeline**: Explore Preprocessing, Lexical Analysis, Syntax Analysis, Semantic Analysis, IR Generation, Optimization, and Target Code Generation.
- **Authentic Specifications**: View real Lex (.l) and Yacc (.y) implementation logic for each phase.
- **Real-time Editor**: Modify C source code and watch the compiler phases react dynamically.
- **Premium UI**: Elegant design featuring Instrument Serif typography, mesh gradients, and smooth GSAP animations.
- **Runtime Simulation**: A built-in terminal that simulates the execution of the generated Stack VM instructions.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, Framer Motion, GSAP
- **Icons**: Lucide React
- **Celebration**: Canvas Confetti

## 📦 Getting Started

1. Clone the repository
2. Install dependencies: \`npm install\`
3. Run locally: \`npm run dev\`
